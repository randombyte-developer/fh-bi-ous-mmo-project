Im Modell Calculator können Parameter (a, b, bits und operation) für die Berechnung angepasst werden.

Im Modell NAND kann useTransistors auf false gestellt werden, damit kein Transistor simuliert wird.
Mit useTransistors=true sind 2 bis 3 Bit empfohlen. Mehr nicht, weil die Simulation sonst zu lange dauert. Mit useTransistors=false können auch 8 Bit simuliert werden.
